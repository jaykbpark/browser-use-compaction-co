whether the compact observation preserved the next browser action while reducing
token load.

observation summary, route, and fallback:

{input}

The span output is JSON with the evaluator result, predicted next action,
compact token estimate, baseline token estimate, and reduction percentage:

{output}

Choose exactly one label:

Prefer the explicit output.passed field when present. Use the predicted and
expected actions only as a fallback if output.passed is missing. In the
explanation, mention the route/fallback and the token reduction when available.
